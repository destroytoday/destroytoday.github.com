JPixel is a PHP web app I coded a long time ago—back when prefixing an app’s name with your first initial was cool. It enlarges images using a series dot patterns. Each pixel is replaced with a 8x8 pixel dot pattern based on the pixel’s greyscale value. I built it in order to generate images that could be printed at a higher resolution, but still retain a degree of sharpness.
