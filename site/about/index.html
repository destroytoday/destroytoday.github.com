About

Jonnie Hallman

I’m tough—I sit in an ergonomic kneeling chair and use a Wacom pen. As a UX developer at Adobe, I write mobile applications in the realm of digital magazine publishing and social networking. Prior to joining Adobe, I attended RISD and MICA, majoring in Graphic Design. While in school, I designed and developed several applications, including DestroyTwitter. In its first three months, DT’s install count reached six figures. It currently sends over one million tweets per month. My day-to-day consists of ping pong, super burritos, and Alex Trebek. I look for jokes in every situation and consistently ruins tear-jerkers. If given a million dollars, I would request it in small, non-sequential, unmarked bills.

destroytoday.com

A venture into new grounds, this site is a first of many for me. It is built on Jekyll, using Haml, Sass, and CoffeeScript preprocessors, as well as Liquid and jQuery for fanciness. It’s completely static—Jekyll compiles every page to static HTML files, so there are none of the security vulnerabilities commonly found in popular CMSs. The pages also load much faster. Along with all the new tech, I’m thrilled to have carried over my app dev practices by source-controlling the site using Git. I also avoid FTP completely by deploying with Beanstalk, which has been a godsend. If any of this is gibberish, definitely check out the links.
